### Особенности реализации эмуляции Кворум-1024 в SpeccyP

## Версия ROM0

Используется версия ПЗУ QU1024V42. Встроенный тест памяти проверяет только 16 страниц (256kiB)

## Расширенная клавиатура

[место для фотографии клавиатуры]

| Клавиши Кворум | PC-клавиатура |
|----------------|---------------|
| `F1`  | `ALT`+`1` |
| `F2`  | `ALT`+`2` |
| `F3`  | `ALT`+`3` |
| `F4`  | `ALT`+`4` |
| `F5`  | `ALT`+`5` |
| `G`   | `ALT`+`6` |
| `RUS` | `ALT`+`9` |
| `LAT` | `ALT`+`0` |

## Поддержка системы CP/M

Образы дисков CP/M имеют расширение .CPM. Формат образа:

1024 байт на сектор, 5 секторов на дорожке, 2 стороны диска, 80 дорожек.

Размер файла 800kiB.
